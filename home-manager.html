快速开始

# 应用配置 (修改后执行)
home-manager switch

# 仅构建 (语法检查 / 预演)
home-manager build

# 构建到隔离 profile (测试用)
home-manager build -b test

# CI 式构建
nix-shell -p home-manager --run 'home-manager build -f ./home.nix'

# 格式化检查
nixpkgs-fmt --check *.nix modules/*.nix

# 自动格式化
nixpkgs-fmt *.nix modules/*.nix

# 临时试用新包
nix-shell -p <pkg>

SSH 私钥管理

所有 SSH 私钥通过 git-crypt 加密存储。磁盘文件始终为明文，Git 历史中为密文。

加密原理

磁盘 (明文)  →  ssh/id_ed25519: OpenSSH private key (SSH 可直接读取)
Git  (密文)  →  git show HEAD:ssh/id_ed25519: binary data (加密 blob)
过滤器       →  .gitattributes: ssh/* filter=git-crypt diff=git-crypt

新机恢复流程

  1. 导入 GPG 密钥：gpg --import /path/to/gpg-backup.asc
  2. 克隆仓库：git clone git@github.com:pengchangg/home-manager-config.git ~/.config/home-manager
  3. 解密 SSH 文件：cd ~/.config/home-manager && git-crypt unlock
  4. 应用配置：home-manager switch
⚠ 注意：SSH 私钥通过 home.activation 以 install -m 600 直接复制，不经过 Nix store（Nix store 强制 0444 权限，SSH 会拒绝）。
GPG 密钥备份：gpg --export-secret-keys --armor E6B06164FE40CC96 > ~/gpg-backup.asc — 失去 GPG 密钥 = 无法解密 SSH 私钥
